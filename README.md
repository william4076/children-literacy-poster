# 儿童识字小报生成器

🎨 基于 AI 的儿童识字小报自动生成工具

## 在线体验

访问：https://william4076.github.io/children-literacy-poster/

## 功能特点

- ✅ 支持 5 个主题：超市、医院、公园、学校、动物园
- ✅ 每个主题包含 15-20 个词汇（带拼音）
- ✅ 自动生成适合 5-9 岁儿童的识字小报
- ✅ 纯前端实现，可部署到 GitHub Pages
- ✅ 支持图片预览和下载

## 使用方法

1. 访问网站
2. 输入你的 [Nano Banana Pro API Key](https://kie.ai/api-key)
3. 选择主题（如：超市）
4. 输入标题（如：走进超市）
5. 点击生成，等待 30-60 秒
6. 预览并下载生成的小报

## 本地运行

直接打开 `index.html` 文件即可使用。

## 技术栈

- 纯前端 HTML + JavaScript
- Nano Banana Pro API
- GitHub Pages 部署
